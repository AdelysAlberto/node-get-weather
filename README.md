# node-get-weather
Api to get weather from Open Weather 

clone repo
run app example = 
```
node app -d "Banfield, Argentina"
```

this return a objetc with weather in celsius, and another info.
```
{
  temp: '32.80',
  feels_like: 306.48,
  temp_min: 303.15,
  temp_max: 307.15,
  pressure: 1011,
  humidity: 32
}
```
